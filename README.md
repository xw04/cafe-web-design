# Energy Cafe - Web Design & Development

## Project Overview
A multi-page website designed for a concept cafe, focusing on user experience, responsive design, and interactive elements.

## Features
- **FAQ Page:** Interactive accordion-style questions and animated backgrounds.
- **Events Page:** Dynamic layout showcasing upcoming cafe events.
- **Shopping Cart:** Functional front-end logic for item selection.
- **Responsive Design:** Optimized for both desktop and mobile viewing.

## Tech Stack
- HTML5
- CSS3 (Animations & Layouts)
- JavaScript (Interactivity)
